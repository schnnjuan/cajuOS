# Contribuindo

Obrigado pelo interesse no CajuOS! Aqui vai como ajudar.

## Sugerir uma tool

Abra uma [issue](https://github.com/schnnjuan/cajuOS/issues/new) descrevendo o problema que você quer resolver. Inclui:

- Qual problema
- Como você resolve hoje (se resolve)
- Exemplo de uso

## Reportar bug

Abra uma [issue](https://github.com/schnnjuan/cajuOS/issues/new) com:

- O que aconteceu
- O que era esperado
- Passos pra reproduzir
- URL onde aconteceu

## Pull request

1. Fork o repo
2. Crie uma branch: `git checkout -b feat/minha-ideia`
3. Faça as alterações
4. Rode `npm run build` — precisa passar limpo
5. Abra o PR descrevendo o que mudou

Orientações:

- Mantenha o escopo pequeno (1 PR = 1 coisa)
- Siga o estilo do código existente
- Teste manualmente antes de abrir
- Links e referências são bem-vindos

## Código de conduta

Seja respeitoso. Assuma boa fé. Isso é um projeto pequeno e pessoal — trate como tal.
