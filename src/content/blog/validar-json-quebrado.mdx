## O problema

JSON colado de log ou resposta de API vem torto: sem quebra de linha, com
vírgula no lugar errado. Debugar na mão é lento.

## A solução

Abra o **JSON Prettify**, cole o conteúdo e ele formata indentado. Se tem
erro de sintaxe, a primeira linha problemática é destacada.

```json
{ "name": "caju", "tools": 52, "open": true }
```

Pronto — copie o resultado formatado de volta pro seu código.
