## O que é uma OG image

Toda vez que você compartilha um link no Twitter/X, LinkedIn, Discord, WhatsApp ou Slack, a plataforma busca uma imagem associada à página. Essa imagem — a **Open Graph image** (OG image) — é o preview visual que aparece junto com o título e descrição do link.

Sem OG image, seu link vira um bloco de texto sem graça. Com uma, ele chama atenção, passa credibilidade e aumenta o **click-through rate (CTR)**.

## Por que isso importa

Estudos mostram que links com OG image têm **até 3x mais cliques** que links sem. Pra projetos novos, landing pages, blog posts ou ferramentas open source, é um dos detalhes de maior impacto com menor esforço.

O problema? Criar uma OG image decente geralmente envolve:

- Abrir Figma/Photoshop
- Escolher fonte, cor, layout
- Exportar no tamanho certo (1200×630)
- Salvar e fazer upload

Isso leva minutos. Ou você pode fazer **em 10 segundos**.

## Como usar o OG Image Generator

1. Abra [cajuos.dev/tools/og-image](/tools/og-image)
2. Digite o título (e subtítulo, se quiser)
3. Escolha uma paleta de cor
4. Clique em **Download PNG**

Pronto. A imagem já sai no tamanho exato (1200×630), com a marca CajuOS no canto.

## Dicas rápidas

- **Título curto (até ~60 caracteres)** funciona melhor — textos longos quebram em duas linhas no preview
- **Subtítulo opcional** — use pra contexto extra tipo o nome do seu projeto
- **5 paletas** disponíveis — Classic (branco limpo), Slate (cinza suave), Warm (tom amarelado), Minimal e Deep (fundos escuros)
- A imagem gerada carrega a marca `cajuos.dev` como watermark — é sutil e vira distribuição orgânica do site
