# Epoch Now — API

Converte timestamp Unix de/para data legível.

## Endpoint

`GET /api/epoch?ts=1737283200`

### Parâmetros

- `ts` (opcional): epoch em segundos. Se omitido, retorna o epoch atual.
- `date` (opcional): data ISO para converter em epoch.
- `tz` (opcional): fuso IANA, ex: `America/Sao_Paulo`.

### Resposta

```json
{
  "epoch": 1737283200,
  "iso": "2026-01-19T00:00:00.000Z",
  "local": "2026-01-19 00:00:00"
}
```
