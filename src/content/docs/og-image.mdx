Gera imagens Open Graph prontas pra compartilhar. Rápido, sem cadastro, sem editor gráfico.

## Como usar

<Command>cajuos.dev/tools/og-image</Command>

1. Acesse a tool
2. Digite o **Título** (obrigatório, até 65 caracteres)
3. Digite o **Subtítulo** (opcional, até 95 caracteres)
4. Escolha uma **paleta**
5. Veja o preview ao vivo
6. Clique em **Download PNG**

A imagem sai em 1200×630 PNG — tamanho exato que Twitter, LinkedIn, Discord e WhatsApp esperam.

## Código

<Command>npx degit schnnjuan/cajuOS/tools/og-image my-og-tool</Command>

```ts
const canvas = document.createElement("canvas");
canvas.width = 1200;
canvas.height = 630;
const ctx = canvas.getContext("2d")!;
ctx.fillStyle = "#ffffff";
ctx.fillRect(0, 0, 1200, 630);
```
