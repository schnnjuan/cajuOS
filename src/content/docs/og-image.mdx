## Como usar

1. Acesse [/tools/og-image](/tools/og-image)
2. Preencha o campo **Título** (obrigatório, até 65 caracteres)
3. Preencha o campo **Subtítulo** (opcional, até 95 caracteres)
4. Selecione uma das 5 **paletas** de cor
5. Visualize o preview ao vivo
6. Clique em **Download PNG**

A imagem gerada tem exatamente 1200×630 pixels, formato PNG.

## Paletas

| Paleta | Fundo | Texto | Indicação |
|---|---|---|---|
| Classic | Branco | Preto | Uso geral, clean |
| Slate | Cinza claro | Preto | Posts sérios, tech |
| Warm | Off-white | Marrom | Conteúdo criativo |
| Minimal | Preto | Branco | Contraste alto |
| Deep | Preto | Roxo claro | Visual moderno |

## Marca d'água

Toda imagem gerada inclui a marca `cajuos.dev` no canto inferior direito. Não é removível — é a forma da ferramenta se distribuir organicamente.

## Limitações

- Sem upload de logo ou imagem de fundo (previsto pra v2)
- Sem API endpoint (previsto pra v2)
- Sem histórico ou login

## Stack

- Canvas API (`toBlob()` pra exportar PNG)
- Next.js 16 (App Router)
- Tailwind v4
- Zero dependências externas
